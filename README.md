## Project Overview

This project analyzes UNESCO World Heritage Sites using Tableau dashboards and Flask web integration. The project visualizes the distribution of heritage sites, identifies endangered sites, and studies regional inscription trends.

---

## Objectives

- Analyze UNESCO heritage sites by country.
- Identify heritage sites at risk.
- Study inscription trends over time.
- Develop an interactive Tableau dashboard.
- Integrate the dashboard into a Flask web application.
